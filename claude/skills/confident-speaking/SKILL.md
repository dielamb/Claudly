# Confident Speaking

> "Structure sets you free. When you have a framework, you can focus on connecting, not remembering." — Matt Abrahams

YOU MUST apply the framework completely. Half-structured communication is unfocused communication.

## Step 1 — Identify the Format (MANDATORY)

YOU MUST select before writing:
- [ ] Full presentation (5-part structure)
- [ ] Status update / quick brief
- [ ] Pitch (60-second)
- [ ] Difficult conversation opener
- [ ] Talking points only

Different formats have different structures. Apply the right one.

## Step 2 — Apply WHAT → SO WHAT → NOW WHAT (MANDATORY at every level)

This applies to each sentence, paragraph, AND the entire piece:

| Part | Question | Non-negotiable |
|------|----------|----------------|
| **WHAT** | What is the information? | State it plainly |
| **SO WHAT** | Why does it matter to THIS audience? | Specific to them, not generic |
| **NOW WHAT** | What action follows? | ONE action, specific, time-bound |

NEVER deliver SO WHAT without NOW WHAT. Information without action is noise.

## Full Presentation Structure (when selected)

All 5 parts are mandatory:

1. **Hook** (30 seconds max)
   - Surprising stat, bold claim, or provocative question
   - NEVER: "Hi, I'm [name] and today I'll be talking about..."

2. **Problem** — make them FEEL it, not just understand it

3. **Solution** — clear, specific, believable
   - One sentence: "We do X, which means Y"

4. **Evidence** — minimum: one data point + one story
   - Specific beats vague every time: "47% faster" not "much faster"

5. **CTA** — ONE action, specific, time-bounded
   - NEVER end with "any questions?" as the closing beat
   - End strong: last words are most remembered

## Pitch — 60 Seconds (when selected)

```
Hook:     [1 sentence — surprising or provocative]
Problem:  [1 sentence — pain in plain language]
Solution: [1 sentence — what you do]
Proof:    [1 concrete result or example]
CTA:      [1 specific ask]
```

Total: 5 sentences. No more.

## Status Update (when selected)

```
WHAT:     [What happened / what we did]
SO WHAT:  [Impact on goal / why it matters to this audience]
NOW WHAT: [Next step / specific ask / decision needed]
```

## Difficult Conversation (when selected)

```
"I want to talk about [specific topic].
Here's what I'm observing: [specific behavior — no adjectives, just facts].
The impact is: [concrete effect on work/team/goal].
What I'd like: [specific ask or question — not a complaint]."
```

## Non-Negotiable Language Rules

- One idea per sentence — always
- "47% faster" not "significantly faster" — always
- Pause instead of filler words ("um," "like," "you know")
- NEVER trail off at the end — last sentence must be strong
