# Microcopy

Every word must earn its place. YOU MUST follow these standards without exception.

## Step 1 — Identify the Context (MANDATORY)

YOU MUST classify before writing:
- [ ] CTA / button
- [ ] Error message
- [ ] Empty state
- [ ] Loading / progress
- [ ] Onboarding
- [ ] Tooltip / helper text
- [ ] Confirmation / success

Different contexts have different rules. NEVER apply generic copy to all.

## Step 2 — Apply Tone Formula (MANDATORY)

**Default:** 80% information, 20% warmth
**Critical moments** (errors, empty states, long waits, onboarding): 70% information, 30% warmth

Warmth = maximum half a sentence, ALWAYS followed by a specific action. Warmth without action is NEVER acceptable.

## Step 3 — Write Using These Rules (ALL mandatory)

**CTAs:**
- Strong verb + specific object. ALWAYS.
- `BAD: "Click here" / "Submit" / "OK"`
- `GOOD: "Create project" / "Save changes" / "Connect account"`
- Max 3 words for primary CTAs

**Error messages — MUST answer all 3:**
1. What happened? (plain language, zero tech jargon)
2. Why? (optional — only include if it helps the user)
3. What now? (MANDATORY — always give a specific next action)

`BAD: "Error 403. Request failed."`
`GOOD: "You don't have access to this file. Ask the owner to share it with you. [Request access]"`

YOU MUST NEVER blame the user.

**Empty states — MUST include all 3:**
1. What they'll see here (context)
2. Why it matters (one sentence max)
3. Primary CTA (specific action)

**Loading:**
- < 2s: spinner only, no text
- 2–10s: "Loading your projects…" (specific, not "Loading…")
- > 10s: progress indicator + what's happening + estimated time

## Step 4 — Consistency Check (MANDATORY)

YOU MUST verify:
- [ ] Terminology is consistent throughout (never mix "Delete"/"Remove" for same action)
- [ ] "Sign in" vs "Log in" — pick one and use it everywhere
- [ ] Action verbs are from the approved set: `Create Connect Run Pause Retry View Edit Remove Save Send Add Open`

## Deliver

Final copy only, formatted by context. No hedging ("you might want to consider..."). Decisive recommendations.
