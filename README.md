# Claude Code Setup

One-command setup for Claude Code with full intelligence stack.

## What you get

- **Claude Code** — Anthropic's AI coding CLI
- **RTK** — Token optimizer (60-90% savings on dev commands)
- **lean-ctx** — Context optimizer MCP (99% token reduction)
- **claude-flow** — Multi-agent orchestration (RuFlo)
- **GSD framework** — Get Shit Done task methodology
- **76 skills & agents** — Pre-configured specialists (memory/intelligence layer)
- **Obsidian vault** — Second brain with AI routing
- **Graphify** — Knowledge graph visualization
- **All hooks pre-wired** — caveman mode, quality gates, memory system

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_USERNAME/claude-setup/main/install.sh | bash
```

Or clone and run:

```bash
git clone https://github.com/YOUR_USERNAME/claude-setup
cd claude-setup
bash install.sh
```

## Requirements

- macOS (Apple Silicon or Intel)
- Internet connection
- ~500MB disk space

## Post-install

1. Open Obsidian → Open vault → `~/Desktop/Labirynt`
2. Start Claude Code: `claude`
3. Verify RTK: `rtk gain`
